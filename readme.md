# Positron Webinar — מבוא ו-Hands-on עם AI ו-Positron

מטרת הריפו הזה היא להוות חומר נלווה לוובינר שמציג עבודה פרודוקטיבית עם Positron, Copilot, Assistant ו-DataBot — ולהוביל משתתפים להמשך מעמיק בתרגול והטמעה.

## תקציר הוובינר

משך: ~60 דקות  

מבנה כללי:

   1. חלק 1 — הצגת הבעיה ו-Positron (הקדמה, למה זה חשוב)
   2. חלק 2 — Live Demo (30 דקות): בניית זרימת עבודה פרודוקטיבית עם Copilot, Assistant ו-DataBot
   3. חלק 3 — הדרן וצעדים הבאים — 45–60 הדקות האחרונות בוובינר (מפורט מטה)
